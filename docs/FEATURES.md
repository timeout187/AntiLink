# AntiLink features

AntiLink is organized around a Discord security workflow: detect threats,
enforce server policy, give moderators visibility, and provide recovery tools.

## Link protection

- Phishing and malicious URL detection
- Discord invite and external-link controls
- Spam-link protection
- Link reputation checks through `/checklink`
- User, domain, role, channel, and category whitelists
- Configurable enforcement and moderation logging

## Server defense

- Emergency Lockdown
- Scam Pattern Guard
- Member Defense for suspicious joins and raids
- Honeypot trap channels
- Member verification
- Anti-Nuke protection
- Server Backup and Restore

Availability varies by plan and server configuration. Review the
[current plan comparison](https://antil.ink/premium) for details.

## Dashboard and visibility

- Discord OAuth authentication
- Per-server security configuration
- Protection, moderation, and audit logs
- Appeals management
- Security analytics
- Public service status and incident reporting
