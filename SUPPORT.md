# AntiLink support

## Before requesting help

Check these public resources:

- [Documentation](https://docs.antil.ink)
- [Command reference](https://antil.ink/commands)
- [Service status](https://status.antil.ink)
- [Dashboard](https://dashboard.antil.ink)

## Get support

Join the [AntiLink support server](https://support.antil.ink) for setup help,
configuration questions, billing assistance, or bug reports.

When reporting a problem, include:

- What you expected to happen
- What happened instead
- The affected command or dashboard page
- Relevant screenshots with private information removed
- The approximate time the issue occurred

Never post bot tokens, passwords, API keys, webhook URLs, payment details, or
other credentials.

## Security reports

Do not report vulnerabilities publicly. Follow
[SECURITY.md](SECURITY.md) for private disclosure instructions.

## GitHub issues

Public GitHub issues may be used for documentation corrections, general bug
reports, and feature suggestions. Support requests that contain private server
information should go through the support server instead.
