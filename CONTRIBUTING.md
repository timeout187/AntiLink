# Contributing

AntiLink's application source code is proprietary and is not developed in this
public repository.

Contributions are welcome for:

- Public documentation corrections
- Broken-link reports
- Clear feature suggestions
- Reproducible public-facing bug reports

Before opening an issue:

1. Check the [documentation](https://docs.antil.ink).
2. Check the [status page](https://status.antil.ink).
3. Search existing issues for the same topic.
4. Remove server IDs, user IDs, tokens, webhook URLs, and private information.

Security vulnerabilities must be reported privately according to
[SECURITY.md](SECURITY.md).
