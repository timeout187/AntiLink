# Getting Started

1. [Add AntiLink to Discord](https://invite.antil.ink).
2. Run `/setup` in your server.
3. Choose a moderation log channel.
4. Review the protection options.
5. Set the bot language with `/language`, which offers 24 choices.
6. Open the [dashboard](https://dashboard.antil.ink) for browser-based control.
7. Configure the dashboard-only modules there. Custom Automod, User Logs, Community
   Health and Analytics have no slash command.
8. Use `/checklink` when you want a reputation check for a URL or domain. A clean
   result means no known threat at the time of the check.
9. To use AI Chat, link an [[AI Membership]] to the server and run `/ai` to set it
   up, including the optional Weekly AI Security Digest.

You need appropriate Discord server permissions to configure AntiLink.

For complete instructions, visit
[docs.antil.ink](https://docs.antil.ink).
